About

I’m a software developer and IT professional with hands-on experience building database-driven web applications and supporting real-world enterprise systems. My work sits at the intersection of development, systems support, and user-focused problem solving.

I have experience designing and developing full-stack web applications using HTML, CSS, JavaScript, Node.js, and MySQL, with a strong focus on structured data, user authentication, and maintainable system architecture. I enjoy working on applications where reliability, clarity, and usability matter as much as visual presentation.

Alongside development, I’ve worked in enterprise IT environments supporting platforms such as Microsoft Dynamics Navision, Salesforce, and Microsoft 365. This has given me practical exposure to system administration, data validation, troubleshooting production issues, and documenting technical processes for long-term maintainability.

One of my key projects, Pulse Link, is a healthcare facility locator platform designed to improve access to medical services through location-aware search, interactive maps, and secure user profiles. Building this system strengthened my understanding of relational databases, geospatial data, API-driven features, and incremental feature development in a growing codebase.

I’m particularly interested in problem-driven and health-tech projects, where technology supports real users and real decisions. I value thoughtful design, clean code, and continuous learning, and I’m motivated by building systems that are practical, scalable, and genuinely useful.
